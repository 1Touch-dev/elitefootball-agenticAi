Frequently asked questions

Quick answers to the support requests we hear most often.

How do I reset my password?

Use the “Forgot password” link on the sign-in page. We’ll send a secure reset link to your email address.

Can I request a refund?

Yes. Refund requests can be submitted within 14 days of purchase from the billing section in your account.

Where can I track my order?

Open your order history to see delivery status, tracking updates, and the latest estimated arrival time.

What should I do if the app is not loading?

Check your internet connection, refresh the page, and make sure your app or browser is up to date.

How do I manage privacy settings?

Visit your account settings to update permissions, data preferences, and account visibility controls.

How can I contact support?

You can contact support through live chat, email, or phone. Response times are shown in the support options below.

const searchInput = document.getElementById('help-search'); const summary = document.getElementById('search-results-summary'); const searchableCards = [ ...document.querySelectorAll('.category-card'), ...document.querySelectorAll('.faq-card') ]; function updateSearch() { const query = searchInput.value.trim().toLowerCase(); let visibleCategories = 0; let visibleFaqs = 0; searchableCards.forEach((card) => { const text = (card.dataset.search + ' ' + card.textContent).toLowerCase(); const isMatch = !query || text.includes(query); card.hidden = !isMatch; if (isMatch && card.classList.contains('category-card')) visibleCategories += 1; if (isMatch && card.classList.contains('faq-card')) visibleFaqs += 1; }); summary.textContent = query ? `Showing ${visibleCategories} categories and ${visibleFaqs} frequently asked questions for “${query}”.` : 'Showing 6 categories and 6 frequently asked questions.'; } searchInput.addEventListener('input', updateSearch);
